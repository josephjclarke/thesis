# Thesis
My Thesis

## Building
Build in draft mode with 

```
make
```

Build in final mode with
```
make final
```

## Cleaning
Use 

```
make clean
```

to remove the nonessential files generated during build.

Use 
```
make deepclean
``` 

to get rid the output pdf as well.

## Linting
Using 
```
make check
```
will check with lacheck
